### User ###
* view/browse ads
    * sort by latest/earliest
    * browse by category
* search ad
    * filter by location/category
* bookmark ad
* create account
* login/logout with account
* manage account
    * edit info/details
    * delete account
* manage ads
    * create ad
    * edit ad
    * remove ad
    * request ad promotion
* compose message
* manage inbox
    * view messages
    * reply to messages
    * delete messages
* report bug issues
* report misused ads
